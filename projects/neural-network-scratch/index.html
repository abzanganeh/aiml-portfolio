Neural Network from Scratch

  • Backpropagation algorithm implementation
  • Multiple activation functions (ReLU, Sigmoid, Tanh)
  • Gradient descent optimization with momentum
  • Regularization techniques (L1/L2, Dropout)

Technologies Used
