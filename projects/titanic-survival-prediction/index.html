Project Overview

This project implements a comprehensive machine learning solution for predicting passenger survival on the Titanic. Using the famous Titanic dataset, I developed and optimized multiple classification models to achieve high prediction accuracy through advanced feature engineering, statistical analysis, and ensemble methods.

Technologies Used

Python Pandas NumPy Scikit-learn Matplotlib Seaborn Jupyter Notebook
View on GitHub Live Demo

Key Features

  • Comprehensive Data Analysis: Detailed exploratory data analysis with statistical insights
  • Advanced Feature Engineering: Created new features from existing data to improve model performance
  • Missing Value Handling: Strategic imputation techniques for optimal data quality
  • Multiple Model Comparison: Evaluated various algorithms including Random Forest, SVM, and Gradient Boosting
  • Ensemble Methods: Combined multiple models for improved prediction accuracy
  • Cross-Validation: Robust model validation to ensure generalizability

Methodology

1. Data Exploration

Performed comprehensive exploratory data analysis to understand the dataset structure, identify patterns, and discover relationships between features and survival rates.

2. Feature Engineering

Created new features including:

  • Family size from SibSp and Parch columns
  • Title extraction from passenger names
  • Age groups for better categorical representation
  • Fare per person calculations

3. Model Development

Implemented and compared multiple machine learning algorithms:

  • Logistic Regression
  • Random Forest Classifier
  • Support Vector Machine
  • Gradient Boosting Classifier
  • Ensemble Voting Classifier

Results

The final ensemble model achieved an accuracy of 84.5% on the test set, with the following performance metrics:

  • Precision: 0.83
  • Recall: 0.79
  • F1-Score: 0.81
  • ROC-AUC: 0.88

Key Insights

  • Passenger class and gender were the strongest predictors of survival
  • Family size had a non-linear relationship with survival probability
  • Fare and age showed significant correlations with survival rates
  • Ensemble methods outperformed individual models by 3-5%
